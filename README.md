# AI4N2S — 小说转剧本系统

基于 Next.js 16 + SQLite + YAML 构建的 AI 辅助小说转剧本数据管理系统。
## 功能

- **小说管理** — 创建/删除小说，上传源文件（PDF、DOCX、TXT）
- **结构化分析** — 将小说文本分析为结构化 JSON（章节、角色、地点、场景）
- **剧本生成** — 从结构化小说数据生成标准剧本格式（场景、对话、转场）
- **剧本编辑** — 可视化编辑场景、角色、对话，支持手动与 AI 辅助
- **YAML 导出** — 剧本支持导出为 YAML 格式，便于人工编辑和版本管理

## 设计理念

小说管理：小说是一个对象，可以包含若干类型，若干个源文件（txt,pdf,docx），以及一组元数据，还有从源文件解析出来的结构化的小说，相当于对小说进行结构统一。
从小说源文件到结构化的小说，定义不同的策略完成解析，这一步主要靠代码，也提供LLM驱动的策略，进行增强
剧本管理：剧本是一个对象，一个剧本可以关联一个小说，也可以不关联。其主要AI功能是从结构化的小说到剧本的映射算法，主要采用LLM驱动的工作流。根据有无配置向量数据库，可选RAG策略是否引入。不引入只用LLM进行驱动，引入则RAG增强，剧本更加完善

## 快速开始

```bash
cd ai4n2s
npm install
npm run seed    # 填充示例数据
npm run dev     # 启动开发服务器
```

打开 http://localhost:3000
